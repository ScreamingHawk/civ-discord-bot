# Civ Discord Bot

A discord bot that notifies when it's your turn in a play by cloud Civilization game

## Usage

Do the following.

### Set Up

Install dependencies

```sh
yarn && cd client && yarn
```

### Start

Single command, this enables *watching*

```sh
yarn dev
```

### Deploy

Set up heroku

```sh
heroku login
heroku create
```

Do the deployment

```sh
git push heroku master
```
